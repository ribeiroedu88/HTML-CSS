Código-fonte

Usando a tag CODE podemos deixar a fonte do código mais espaçadas facilitando a leitura ex.:
document.getElement('teste') comando sem a tag CODE
document.getElement('teste') comando com a tag CODE

Para manter a formatação de código podemoe usar junta com a tag CODE a tag PRE que mantem a formação de código fonte ex.:

Código sem a tag PRE

num = int(input{'digite um numero'}) if num == 0; print 'num ' else: prnt ' vazio'

Código com a tag PRE

        
    num = int(input{'digite um numero'})
        if num == 0;
            print 'num '
        else:
            prnt ' vazio'
